# QWaveBox

一款基于Qt和FFmpeg的音视频播放器。

## 功能特点

- 播放本地音频和视频文件
- 播放网络媒体资源
- 支持播放列表管理
- 提供播放速度控制
- 音量调节和静音功能
- 进度条拖动实现快进快退
- 支持全屏播放

## 界面布局

- 左侧为播放区域：显示视频内容和播放控制器
- 右侧为播放列表：提供多个标签页
  - 当前播放列表
  - 本地媒体列表
  - 网络媒体列表
  - 收藏列表

## 技术栈

- Qt 5.15.2+：用于UI界面构建
- FFmpeg：用于音视频解码
- C++17：编程语言
